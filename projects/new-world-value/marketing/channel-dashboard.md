## 🔍 How to Check Each Channel (Weekly)

### Substack (5 min)
1. Go to **substack.com** → Dashboard → **Analytics**
2. Note: Subscribers, Open Rate, Click Rate
3. Copy numbers to tracker above

### X.com (3 min)
1. Go to **x.com/analytics**
2. Select last 7 days
3. Note: Impressions, Profile Clicks, New Followers
4. Copy post URL from GenSpark confirmation

### Instagram (3 min)
1. Go to **Instagram** → Profile → **Insights**
2. Select last 7 days
3. Note: Reach, Profile Visits, New Followers
4. Copy post URL from GenSpark confirmation

### Quora (5 min)
1. Go to **quora.com/analytics**
2. Note: Total views, upvotes (last 7 days)
3. Click each answer → copy view count
4. Paste answer URLs in tracker

### WhatsApp (1 min)
1. Check broadcast replies
2. Count responses
3. Note in tracker

### Telegram (1 min)
1. Check channel subscriber count
2. Note post views
3. Update tracker

### bitly (2 min)
1. Go to **bitly.com/a/links**
2. Click each link → view stats
3. Note: Total clicks, top source, top country

**Total Time:** ~20 minutes every Thursday
