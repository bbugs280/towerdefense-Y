```
### Task 2: Post to Instagram

Create a Story/Reel post with the following caption:

```

```
**Requirements:**
- Use existing NWV branding/visuals (or create simple text graphic)

- Include "Link in bio" CTA
- Post from @newworldvalue account
- Confirm post is live and share the post URL

---

### Task 3: Answer Questions on Quora

For each of the 3 Quora question URLs provided above:

1. Read the question and top answers
2. Write a 300-400 word answer that provides genuine value:
   - Start with a direct 1-2 sentence answer
   - Include 2-3 data points or insights
   - Share a framework or perspective
   - Soft CTA at the end: "I wrote a full analysis on this with [X] scenarios + historical precedents. Free to read: [bitly Read Link]"
   - End with: "Happy to discuss — what aspect worries you most?"

3. Post the answer
4. Confirm each answer is live and share the answer URLs

**Important:**
- Do NOT spam — each answer must be unique and provide real value
- Vary the wording across the 3 answers
- If a question already has 50+ answers, skip it and find a similar question with fewer answers
```

## ⚠️ Troubleshooting

| Issue | Solution |
|-------|----------|
| GenSpark can't log in | Re-authenticate in GenSpark settings → Credentials |
| X post too long | Shorten hook — each data line under 50 chars |
| IG post rejected | Check image format (1080x1920 for Reels/Story) |
| Quora answer flagged | Add more original analysis, reduce link prominence |
| GenSpark hangs | Refresh, re-paste prompt, try again |
