# New World Value — Product Requirements Document

## Overview

**Product:** Human-led, AI-augmented signal service  
**Mission:** Map the emerging value system of 2026+ — helping individuals recalibrate from old metrics (GDP, home ownership, career ladder) to new ones (Security, Meaning, Autonomy, Clarity, Resilience)  
**Core Question:** "What is enough?"  
**Channels:** WhatsApp (primary) + Telegram (secondary) + Substack (tertiary/archive)  
**Cadence:** 1 essay/week + 3-5 short hooks/week  
**Target Market:** English-speaking professionals in Hong Kong + Asia (25-45)

## The Five Values (Pillars)

Each essay explores how one of these values is being redefined in the new era:

| Value | Old Definition | New Definition | Example Essay |
|-------|----------------|----------------|---------------|
| **Security** | Home ownership, stable job, savings account | Energy resilience, learning agility, exit options, multiple income streams | "Your home is no longer your security. What is?" |
| **Meaning** | Career ladder, status, titles | Enough, autonomy, boundaries, non-negotiables | "The promotion won't make you happy. This might." |
| **Autonomy** | Financial independence, early retirement | Control over time, attention, and energy — right now | "You don't need FIRE. You need 2 hours back per day." |
| **Clarity** | Expert opinions, news consumption, credentials | Personal verification, signal filtering, inner compass | "The world isn't chaotic. It's over-connected. Here's how to filter." |
| **Resilience** | Emergency fund, insurance, backup plans | Adaptive capacity, local networks, skill redundancy | "Resilience isn't what you store. It's what you can rebuild." |

## User Stories

### As a Reader

| ID | Story | Acceptance Criteria |
|----|-------|---------------------|
| US1 | As a busy professional, I want to understand what value is shifting so I can recalibrate my own life | Each essay focuses on 1 value. Clear "old vs. new" contrast. Actionable takeaway. |
| US2 | As a skeptic of mainstream narratives, I want data-grounded analysis without ideology | Every claim cites a public source. No opinion presented as fact. |
| US3 | As a WhatsApp user, I want weekly signals pushed to my phone | WhatsApp broadcast sent within 24 hrs of essay publish. |
| US4 | As a sharer, I want forwardable content | Each essay includes 1-2 "tweetable" lines. First 3 essays free. |
| US5 | As a privacy-conscious reader, I want to consume without being tracked | No personal data collected. Anonymous subscribe option. |

## Content Framework

### Essay Structure (500-800 words, <5 min read)

1. **Hook** — Human moment or specific number (not abstract concept)
2. **Old Value** — What used to define this value
3. **Why It's Failing** — Data + observable behavior showing the shift
4. **New Value** — What's emerging instead
5. **What This Means for You** — 1 specific action or question
6. **[VINCENT EDIT]** — Personal observation that grounds it in reality

### Weekly Cadence (~30-40 min Vincent time)

| Step | Tool | Time | Owner |
|------|------|------|-------|
| 1. AI pulls latest data from pre-defined sources | AI + web_search | 10-15 min | AI |
| 2. AI drafts essay from value framework | AI | 5-10 min | AI |
| 3. Vincent reads + tweaks tone + adds personal observation | Manual | 15-20 min | Vincent |
| 4. Vincent spot-checks 2-3 key stats | Manual | 5-10 min | Vincent |
| 5. Publish to WhatsApp + Telegram + Substack | Manual | 5 min | Vincent |
| **Total** | | **~35-50 min/week** | |

## Is It Still Worth Promoting?

### Yes — With These Conditions

**Why YES:**

| Factor | Assessment |
|--------|------------|
| **Market gap** | No Asia-grounded value system newsletter exists. US/EU voices dominate. |
| **Channel fit** | WhatsApp penetration in HK (~90%) is far higher than Substack (~5%). This is an advantage, not a weakness. |
| **Time investment** | ~30-40 min/week is sustainable. Low risk. |
| **Differentiation** | Values framework (Security, Meaning, Autonomy, Clarity, Resilience) is unique. Not just news analysis. |
| **Personal brand** | Builds Vincent's thought leadership without personal exposure. |
| **Optionality** | If it works: monetization, community, speaking opportunities. If it fails: 8 weeks of content, lessons learned. |

**Why CAUTIOUS:**

| Factor | Assessment |
|--------|------------|
| **Cold start** | First 100 subs will require direct invites. No passive discovery. |
| **HK audience size** | English-speaking professionals 25-45 in HK: ~2.5M addressable, but <10% likely to engage. |
| **Monetization ceiling** | HK newsletter market is smaller than US. Paid conversion likely <5%. |
| **Regulatory perception** | Geopolitical content in HK carries some risk (mitigated by public-data-only approach). |

## Final Recommendation

**PROMOTE — with revised expectations:**

| Expectation | Original | Revised |
|-------------|----------|---------|
| Primary channel | Substack | WhatsApp |
| Week 8 subs | 200+ (mostly Substack) | 200+ (mostly WhatsApp) |
| Monetization potential | Paid Substack tiers | Donations, sponsorships, community |
| Time to traction | 8 weeks | 8-12 weeks (slower but more sustainable) |
| Success definition | Growth + engagement | Engagement + forward rate (not just subs) |

**Decision framework:**

- If you want **fast growth + monetization** → This is not the right project.
- If you want **thought leadership + audience building + low time commitment** → This is a strong fit.
- If you want **to test a unique voice in an underserved market** → This is worth 8 weeks.

**Kill switch:** Week 8 review. If WhatsApp forward rate <5% and reply rate <2%, the content isn't resonating. Kill without guilt.
